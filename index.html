Global Emotional Index (GEI)

A neutral, non-evaluative global reference indicator

Global Emotional Index (GEI) provides aggregated emotional pressure context at a global level. It is designed to exist alongside economic, social, and geopolitical indicators as a contextual reference only.

About GEI

Global Emotional Index (GEI) is an experimental informational indicator created to reflect aggregated global emotional pressure trends.

GEI does not explain, predict, interpret, or influence individual or collective behavior. It does not assess countries, populations, governments, or institutions.

How to Read the Index

GEI is centered around a neutral baseline. Movements above or below this baseline indicate relative changes in aggregated emotional pressure.

  • Values above baseline indicate reduced collective pressure.
  • Values below baseline indicate increased collective pressure.

Positive individual emotional states may coexist with elevated collective pressure. GEI reflects systemic context, not personal experience.

Methodology Overview

GEI uses a stability-focused, baseline-centered methodology designed for consistency and comparability over time.

  • Stable neutral baseline reference
  • Aggregated global contextual signals
  • Optional voluntary anonymous inputs (low-weighted)

The methodology prioritizes long-term signal stability over short-term sentiment responsiveness.
